Gestion Empresarial-Proyecto Universitario



Descripcion

Este sistema es una aplicacion de gestion empresarial desarrollada en c# que permite administrar las operaciones principales de una organizacion, integrando una base de datos normalizada a la Tercera Forma Normal (3FN).

Tecnologias Utilizadas

Lenguaje. C# (.Net)
Base de Datos. SQL Server (Relacional)
Normalizacion. 3ra Forma Normal (3FN)
Control de Versiones. Github

Estructura del Repositorio

/src: Contiene el codigo fuente de la aplicacion.
/database: Incluye el script de creacion .sql y el Diagrama de Entidad Relacional (DER).
/ Contiene el archivo de la Tesis, memoria tecnica del proyecto.

Base de Datos (3FN)

La base de datos fue diseñada para la eliminacion de redundancias y proteger la integridad de datos.

1FN. Eliminacion de grupos repetidos.
2FN. Eliminacion de dependencias parciales.
3FN. Eliminacion de dependencias transitivas.

Nota para el profesor: El diagrama de relacion se encuentra adjunto en la carpeta /database para su revision.
